# Reasoning Style: Standard Dispatch

This is the default reasoning style. It matches the standard agent-dispatch procedure
with no modifications. Each expert persona analyzes independently in Phase 1, the
cross-critic identifies gaps and contradictions in Phase 2, and the facilitator
synthesizes in Phase 3.

No phase injections — this style is a no-op that preserves current behavior.
