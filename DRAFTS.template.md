# Drafts Manifest

This repo has unpromoted drafts generated by CWOS engines. Each draft is the
engine's first-pass inference from repo content — the founder reviews and
promotes (or edits, or rejects) before the artifact becomes load-bearing.

**Why drafts:** per CWOS's `feedback_no_silent_install_no_user_invention`
principle, engines never silently inject load-bearing artifacts into a repo.
They draft to a `.draft.md` (or `.draft.<ext>`) file the founder reviews;
promotion happens by rename only. This file is the index.

---

## How to promote

1. Open the draft file listed below.
2. Review against the per-engine checklist (link in each section).
3. Edit as needed — the draft is yours to revise.
4. Rename: `mv <path>.draft.<ext> <path>.<ext>`
5. The next engine run that depends on the artifact will pick up the promoted version.

To reject a draft entirely: delete the `.draft.md` file. The engine will
re-draft on the next run unless the underlying program is set to skip Phase 0b.
