## Phase 2: Hook Consolidation

### Goal
Merge redundant hooks to reduce friction and simplify the enforcement layer.

### Analysis

**Redundant Hooks (Write/Edit):**

1. `write-gate.cjs` (PRIMARY - KEEP)
   - Checks bypass file
   - Checks exempt files
   - Checks protected files
   - Finds task in backlog
   - Checks task assignment
   - BLOCKS if conditions fail

2. `permission-request-handler.cjs` (REDUNDANT - MERGE)
   - Also loads backlog
   - Also checks task assignment
   - Provides more verbose output
   - Merge useful output into write-gate

3. `decomposition-validator.cjs` (KEEP - different purpose)
   - Validates ///LOCAL_TASKS JSON format
   - Warns on large changes >100 lines
   - Does NOT duplicate task lookup

### Changes

| Action | File | Description |
|--------|------|-------------|
| MERGE | `permission-request-handler.cjs` → `write-gate.cjs` | Move verbose error messages |
| DELETE | `permission-request-handler.cjs` | Remove after merge |
