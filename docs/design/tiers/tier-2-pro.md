# Tier 2 — Pro / Grinder

**ID:** `tier-2-pro`
**Role:** Advanced features for serious and pro players
**Target $/mo (placeholder):** ~$49
**Last reviewed:** 2026-04-21

---

## Who this tier is for

- Primary: [Rounder](../personas/core/rounder.md), [Multi-Tabler](../personas/core/multi-tabler.md), [Online MTT Shark](../personas/core/online-mtt-shark.md), [Circuit Grinder](../personas/core/circuit-grinder.md), [Hybrid](../personas/core/hybrid-semi-pro.md), [Analyst](../personas/core/analyst-api-user.md), [Traveler](../personas/core/traveler.md)
- Secondary: ambitious Apprentice, serious Scholar

## Role vs. adjacent tiers

- Upgrade from: Plus (when exploit depth, sidebar, ICM, or advanced modeling becomes essential)
- Upgrade to: Studio (when coaching, staking, team, or API is needed)
- Reason to upgrade to Studio: multi-user role, API access, or team/group billing

---

## What's included

### Main app

- Everything in Plus, plus:
- Full Bayesian villain modeling with game-tree EV (depth-2, depth-3 within time budget)
- Advanced exploit engine with blockers, fold-curves, sizing tells
- ICM tools (adjusted EV at bubble/FT)
- Bounty-adjusted EV mode
- Satellite / seat-bubble strategy switch
- Custom drills from own hand history
- Multi-device sync (unlimited devices)
- Advanced filters + similar-spot search across history
- Player analysis + weakness detection + villain profile depth
- Multi-currency bankroll + FX
- Time-zone-aware session boundaries
- PT4 / HM3 hand-history import

### Sidebar extension

- Full sidebar with all advanced features (fold curves, depth-3, advanced tells).
- Requires separate Ignition / site setup.

### Platform-wide

- As Plus.

## What's NOT included (moves you to Studio)

- Coach dashboard with student queue
- Staker portal with horse read-access
- Multi-user team billing
- API / webhooks
- Verifiable / signed sessions

## Caps and limits

- Player database: unlimited
- History window: unlimited
- Device count: unlimited
- Sync: included (main app + sidebar)

---

## Typical customer journey

- Rounder upgrades from Plus after trial of Bayesian villain modeling on home casino regs.
- Multi-Tabler starts at Pro directly (or Sidebar Lite) — Plus doesn't serve their use case.
- Online MTT Shark upgrades when ICM / bounty features prove out ROI.
- Circuit Grinder upgrades before first WSOP to test Day-1/Day-2 continuity.

## Retention risks

- Exploit engine advice quality not materially better than Plus → churn.
- Sidebar flaky → churn (sidebar is the primary surface for Multi-Tabler and Shark).
- Competitor bundles similar features cheaper.
