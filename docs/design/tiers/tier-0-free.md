# Tier 0 — Free / Tourist

**ID:** `tier-0-free`
**Role:** Evaluation + light recreational use
**Target $/mo (placeholder):** $0
**Last reviewed:** 2026-04-21

---

## Who this tier is for

- Primary: [Newcomer](../personas/core/newcomer.md), [Weekend Warrior](../personas/core/weekend-warrior.md) (initial), [Scholar](../personas/core/scholar-drills-only.md) (casual)
- Secondary: free-evaluation path for any persona before deciding on Plus or Pro

## Role vs. adjacent tiers

- Upgrade from: none (entry point)
- Upgrade to: Plus (Regular) once user hits caps or wants full exploit engine / sidebar
- Reason to upgrade: more than 50 tracked players, history longer than 7 days, or wants exploit advice depth

---

## What's included

### Main app

- Hand entry (live)
- Basic session P&L
- Seat context menu + basic player assignment
- Session list (last 7 days)
- Showdown resolution
- Settings + account
- Tournament Dashboard (basic — blind timer, stacks; no ICM)
- Preflop drills (Explorer mode only)
- Basic mid-hand action hint (not the full exploit engine)

### Sidebar extension

- Not available at this tier.

### Platform-wide

- Offline-first, local-mode operation
- Account recovery
- Onboarding tour
- Accessibility modes
- Data export of own data (CSV)

## What's NOT included

- Full exploit engine (Layer 2/3 depth)
- Villain Bayesian modeling beyond sample-based baseline
- Sidebar extension
- Advanced drills (Postflop full suite, custom drills)
- Multi-device sync
- Hand replay
- Player analysis / weakness detection

## Caps and limits

- Player database: 50 players
- History window: 7 days
- Device count: 1
- Sync: none (local-only)

---

## Typical customer journey

- Newcomer downloads → opens Table View → logs first session.
- Weekend Warrior plays 3 weeks → hits 50-player cap or wants month-over-month history → evaluates Plus.
- Scholar opens Preflop Drills → completes Explorer → wants full library → evaluates Plus or stays free.

## Retention risks

- Not enough value in the free tier → uninstalls.
- Too much value in the free tier → never upgrades.
- Common complaint: "it logged my hands but didn't tell me anything useful" — risk if Layer-1 exploit hints are removed.
