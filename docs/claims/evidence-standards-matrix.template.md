## How to review

For each domain:

1. **Tier definitions are accurate** for your domain's professional norms (e.g., clinical T1 = peer-reviewed primary literature; legal T1 = controlling case law or statute).
2. **Claim-type-to-tier mappings are correct** — does each claim type in your register really need the tier you've mapped it to?
3. **Edge cases addressed** — are there claim types that span tiers (e.g., "anatomy" can be textbook for settled regions but peer-reviewed for active research)?

The matrix is what makes the difference between a claims register that's *audited* and one that's *audited well*.
